# Iris Flower Classification

## Overview
This project uses Machine Learning to classify Iris flowers into three species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

## Dataset
The dataset contains 150 flower samples with the following features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Technologies Used
- Python
- Pandas
- Scikit-Learn

## Algorithm
Decision Tree Classifier

## Steps
1. Load dataset
2. Select features and target
3. Split data into training and testing sets
4. Train Decision Tree model
5. Predict flower species
6. Evaluate accuracy

## Results
The model achieves approximately 95-100% accuracy depending on the train-test split.
